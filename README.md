
# 一、它是什么？
它是一款用来`合租租友`进行`生活开销记账`的小工具，小到只有5M。上手容易，简单轻便！

它支持：
- 注册和登录，以每个家庭为单位；
- 可进行一些简单的`日常开销录入`、`每月结算`和`历史开销查看`功能。

> * 使用H5+HbuilderX，用了三天时间在朋友已有基础上开发而来；
> * 全前端，没有后台服务，数据存储为在线的`LeanCloud`（代码较渣，毕竟笔者只是个弱鸡后端）。

感谢 @yangshare 的源码共享（主要功劳）！
